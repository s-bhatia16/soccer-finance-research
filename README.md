# Finances of European Soccer Teams
Research Report

## Overview
This research report analyzes the financial impact of COVID-19 on European soccer clubs, with a focus on how the pandemic affected core revenue streams including broadcasting income, matchday revenue, and commercial operations. The goal of the project is to translate financial data into clear, decision-oriented insights.

## Key Questions
- How did COVID-19 disrupt revenue composition across European soccer clubs?
- Which revenue streams were most vulnerable during lockdown periods?
- How did financial exposure differ across clubs and leagues?

## Methodology
- Collected and structured financial data using Excel and SQL
- Analyzed year-over-year changes in revenue streams pre- and post-COVID
- Identified trends in revenue loss, recovery patterns, and structural dependence on matchday income
- Synthesized quantitative findings into a written research report

## Tools & Technologies
- Excel — data cleaning, aggregation, and exploratory analysis
- SQL — structured querying and revenue breakdown analysis
